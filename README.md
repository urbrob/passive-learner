# Project created by students for students

## Git workflow
1. Checkout to master
2. Create new branch based on master `fix/bug-from-something` or `feature/cool-new-feature`
3. Create and add your changes 
4. Push it to origin
5. Create PR
6. Ask for review

## Technologies
- API 
  - Python
  - Django

- APP
  - Kotlin

- DevOps
  - Docker
